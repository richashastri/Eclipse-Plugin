Sample example of the PiVi project

Let us see how a real world problem can be modelled in PiVi and how Java code is generated from the visual representation.

  1. Drag and drop the Start block from the palette to the main Canvas to mark the start of the main block the code.
  2. Select the instruction block and create an instance of the thread.
  3. Make a connection from the start node to the instruction node using the connection start link.
  4. Select the thread start block to provide instructions to be run in the thread.
  5. Select instruction block connected to the thread start block and write the instructions.
  6. Select the print block and write the value of the variable to be printed.
  7. Select synchronization start block and connect it to the instruction and the print block followed by Synchronization end block. This would indicate that the instructions will run in a synchronized manner.
  8. Select the connnection links and make the necessary series of connections.
  9. Click on the generate code button to generate the Java code.
  10. Refresh the project and you will see a Solutions.java file is generated which will contain the syntactically correct java code.
  11. The user can run the Solutions.java file to execute and to get the desired output.
