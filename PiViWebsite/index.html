PiVi - An Eclipse Plugin for visual programming

1. Introduction

  • The Pivi aims at providing an easier and interactive way to create a visual model of a programming construct like if-else, while, parallel programming and other instructions.
  • This Eclipse Plug-In is focused on facilitating teaching/learning techniques and implications of parallel, concurrent and multicore programming.
  • The plugin presents a palette of icons like if-else, instructions, connections, while, parallelization that can be used to create a program visually.

  • Each icon in the palette represents different constructs of a programming language. The plugin allows users to connect the icons present on the canvas by validating the connections.
  • The user is able to modify the parameters provided for each icon. Concurrent programming features are provided by the plugin.
  • The plugin generates the code in Java programming language which can be understood by the user in an effective way and can be executed to get the solution to the user problem statement.

3. Building components of PiVi

  • Tools used
    • Ecplise IDE
    Java 8
    GMF (Graphical Modelling Framework)
    EMF (Eclipse Modelling Framework)
    XML parser
    Git

  • Parts of PiVi

    • The PiVi development environment is made up of two basic parts:

    Palette:
    It consists of all the nodes and connection link icons
    Chart
    This is the area where the user is able to drag and drop nodes and add connections amongst these nodes.

  • Basic components of PiVi
    • Node:
    It can be defined as an icon in the UI, which symbolizes a programming construct of Java.
    Connection Link:
    It is a valid link between any two nodes selected that depicts the flow of the program
    Generate Code Button:
    It is a button which helps to generate a Java code, following the visual representation of the code.

4. Types of Block of PiVi


Start

This icon represents the start of the main method in the program.

Sync Start

This icon represents the start of synchronization of a method or an instruction set.

Sync End

This icon represents the end of of synchronization of a method or an instruction set.

Instruction

It is a block through which any instruction statement can be entered in the program.

Print Statement

It is the block through which user can add the output statements to the program.

If Start

This icon represents the start of the If loop and takes the condition as the input parameter.

If End

This icon represents the end of the If loop.

While Start

This icon represents the start of the looping statement where looping condition can be given.

While End

This icon represents the end of the While loop.

Method Start

This icon represent the start of the user defined anonymous method.

Method End

This icon represents the end of the user defined anonymous method.

Thread Start

This icon is used to start the implementation of any function inside the thread.

Thread End

This icon represents the end of the function which is executed inside the thread.

5. Steps to create a PiVi project

Steps to create a PiVi project are as follows, after successful download of the plugin.

  1. Open Eclipse IDE and create a new General project and name it.
  2. Select the type of the project as - PiVi Diagram from Examples folder.
  3. A PiVi diargram environment would be created which will consist of Canvas and Palette, which will consist of Nodes and connnection links associated with it.
  4. A PiVi XML file and PiVi diagram file will get generated. The PiVi digram file is the Graphical interface part.
